# Processed data

`iris_processed.csv` is generated by the ETL pipeline. It contains the four
canonical numeric flower measurements and the normalized `species` target.

The pipeline rejects incomplete, non-finite or non-positive measurements and
unsupported class labels, and removes exact duplicate model rows. It does not
calculate preprocessing statistics before the train/test split;
`StandardScaler` is fitted only on training data inside the scikit-learn model
pipeline.
